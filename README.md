## teenytorch

Tiny utilities for working with PyTorch.

### Installation

```bash
pip install teenytorch
```

For local development:

```bash
pip install -e .
```

### Usage

```python
import teenytorch
print(teenytorch.__version__)
```

### Development

- Python 3.9+
- Build backend: Hatchling

Run tests (requires pytest):

```bash
pytest -q
```
